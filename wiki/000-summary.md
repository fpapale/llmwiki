## Signal Processing for Telecommunications and Economics Lab  
**Course:** Machine Learning and Data Processing (AA24/25) – Economics & Big Data  
**Instructor:** Prof. Ing. F. Benedetto (Chair, IEEE 1900.1)  
**Academic year:** 2024‑2025  

### How to enrol
1. Log‑in to **E‑learning Economia** (https://economia.el.uniroma3.it/) with university credentials.  
2. Select the course **“AA24/25 – Machine Learning and Data Processing”**.  
3. All course resources (info, material, announcements) are on the Moodle / MS‑Teams page.

### Curriculum (topics covered)
- Foundations of AI & machine learning  
- Supervised learning: regression & classification  
- Core supervised algorithms (e.g., linear models, SVM, decision trees)  
- Core unsupervised algorithms (e.g., clustering, dimensionality reduction)  
- Neural networks & deep learning  
- Data preprocessing, selection & transformation  
- Natural Language Processing (NLP)  
- Explainable AI (XAI)

### Assessment
- **Written exam** (two parts):  
  1. Theory questions & exercises  
  2. Python/Matlab programming tasks  
- **Exam registration** only through **GOMP** (https://gomp.uniroma3.it/).  
- **On exam day** bring: ID, student number, and E‑learning credentials.
